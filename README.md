# Azure-Infrastructure-Operations-Dashboard
A dashboard with real time logs &amp; cost visibility.

Overview

This project demonstrates the design and implementation of an Azure Operations Dashboard used to monitor and manage cloud infrastructure.

The dashboard provides visibility into:

- Virtual Machine Health
- Azure Monitor Metrics
- Backup and Recovery
- Alert Management
- Networking
- Resource Governance

## Technologies Used

- Microsoft Azure
- Azure Virtual Machines
- Azure Monitor
- Azure Backup
- Azure Site Recovery
- Recovery Services Vault
- Azure Networking
- Azure Dashboards

## Key Outcomes

- Improved infrastructure visibility
- Centralised monitoring
- Backup compliance monitoring
- Alert management
- Resource governance reporting
